## Run the code

1. Clone the folder
2. Install the requirements 
    <code> pip install -r requirements.txt </code>
3. Start the uvicorn server
    <code> uvicorn main:app --host "0.0.0.0" --port 8000 --reload </code>
4. Run the index.html
